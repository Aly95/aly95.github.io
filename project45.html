Project 45

Project 45 utilises Google's Firebase and a variety of API's to present users with blogs, user created events, news articles and an online shop.

In order to maintain user accounts in a safe and secure manner the application currently uses Firebase email and password authentication, with the addition of Facebook authentication coming in the future.

Events can be sorted by most popular, based on the number of users currently attending, and also by distance, using the distance between the devices current GPS coordinates and the events location coordinates.

Kodein has been used for dependency injection and MVVM as the design pattern. GitHub access to the project will be given on request
