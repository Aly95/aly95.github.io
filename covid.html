CovidCheck

This project was completed within the given timescale of a week, as part of a challenge set by TopCoder.

The application features three pages, one for total coronavirus numbers, one for a list of countries individual coronavirus statistics and the final integrates Google maps with clickable markers which brings up a bottomsheet with the countries statistics on.

The application also allows the user to take a screenshot of the current page and share it using androids standard share dialog.

The code for this project can be found on GitHub
